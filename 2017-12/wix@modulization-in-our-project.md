# 工程化在项目中的实践

## Webpack Babel 模块化

### 什么是模块化

  抽象
  抽象事物的能力
  ``` text
  EG:
  学英语->主干句
  理科=>具象（逻辑）
  写程序亦如是
  ```

### 为什么要用模块

* 作用域封装
* 重用性
* 解除耦合
* 按需加载

## 模块化发展史

* script标签
* 立即执行函数表达式
* AMD
* CommonJS 与 Nodejs 模块系统
* *ES6 Module*
